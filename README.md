# Running locally

This is powered by Gatsby.

Pre-requisites:

* [Node.js](https://www.gatsbyjs.com/docs/tutorial/part-0/#nodejs) (v14.15 or newer)
* Gatsby CLI - run `npm install -g gatsby-cli`

Run site:

* Clone repo
* Run `yarn install` to install dependencies
* Run `gatsby develop`
